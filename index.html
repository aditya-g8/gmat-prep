Suggested daily study loop (~60–75 min)

1
Review concept (15–20 min)
Open Quant / DI / Verbal, read 1–2 topics, try examples before revealing, use the interactive tools, flip cards, mark learned.
2
Drill matching questions (30–40 min)
Open Practice, filter to the same question type, run a timed set in Exam mode. Then switch to Study mode to read every explanation.
3
Review error log (10–15 min)
Highest-leverage step. Open the error log, read "why likely wrong" for each miss, then run "Drill my mistakes." Cleared once you get them right.
4
Reference as needed
Keep the Cheat Sheet or Handbook open for formula lookups. Use the in-page formula boxes without leaving the concept tutor.
Two things to know. (1) Progress (learned topics, error log, session scores) saves in this browser on this device — it does not sync. Pick one main device for tracked practice. (2) Three DI question types (Graphics Interpretation, Multi-Source Reasoning, Table Analysis) are available only in the publisher's Online Question Bank at mba.com (access code inside your Official Guide).
